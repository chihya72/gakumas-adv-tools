## 🛠️ 工具功能

### Parser（解析器）
- 解析 Unity ADV 原始脚本（.txt）
- 导出为结构化 JSON 格式
- 支持批量处理 2962+ 个脚本文件

### Editor（编辑器）
- 可视化 ADV 脚本编辑界面
- 实时预览对话和时间轴
- 资源选择器集成

### Database（资源数据库）
- 角色模型、动作、场景资源管理
- RESTful API 服务
- 前端组件集成

### Analyzer（分析工具）
- 面部表情 Index 使用频率统计
- 组合表情模式分析
- 实验指南生成
